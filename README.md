# 🏏 CricStat AI: Next-Gen Sports Analytics Dashboard

**CricStat AI** is a high-fidelity sports analytics platform designed to revolutionize the fan experience and broadcast commentary using the power of **Gemini AI**. Built for high-pressure match scenarios, it provides real-time tactical insights, community sentiment tracking, and predictive match-momentum metrics.

🚀 **Live Demo**: [https://cricstat-9a04c.web.app](https://cricstat-9a04c.web.app)

---

## ✨ Key Features

### 1. ⚡ Tension Engine (AI Wicket Prediction)
*   **Dynamic Pressure Score**: Uses a proprietary algorithm to calculate match tension (1-10) based on run rate, wickets, and dot-ball pressure.
*   **Wicket Probability**: Real-time prediction of an upcoming wicket using simulated TF.js logic.
*   **Visual Feedback**: Pulse animations and "Flash Danger" alerts for high-stakes moments.

### 2. 🗳️ Poll Aggregator (Community Oracle)
*   **Live Engagement**: Real-time fan voting system to predict match outcomes.
*   **Instant Aggregation**: Visual bars and percentages update globally as votes are cast.
*   **Match Pulse**: Syncs community sentiment directly with AI win probabilities.

### 3. 🧠 Gemini Strat-Insights
*   **Contextual Analysis**: Generates deep tactical cards based on live match events (Wickets, Boundaries, Pressure).
*   **Dynamic Content**: Strategy shifts in real-time—from defensive 4th-stump analysis to aggressive death-overs tactics.

### 4. 🎙️ AI Multi-Modal Voice Commentary
*   **Audio Insights**: Built-in AI voice assistant reads tactical recommendations aloud.
*   **Accessibility**: Ensures complex match data is accessible through audio-visual feedback.

### 5. 📊 Real-time Match Simulator
*   **Live Demo Mode**: Integrated backend simulation allows for high-fidelity demonstrations without external API dependencies.
*   **Full Sync**: Scorecard, Overs, Partnerships, and AI Insights all update in perfect synchronization.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v20+)
*   Angular CLI (`npm install -g @angular/cli`)
*   Firebase CLI (`npm install -g firebase-tools`)

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/atharvachare/ExtremeCoders-GDG-Nashik.git
    ```
2.  Navigate to the project folder:
    ```bash
    cd cricstate
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Run locally:
    ```bash
    npm start
    ```
